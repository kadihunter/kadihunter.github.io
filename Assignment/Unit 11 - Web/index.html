Summary: Lattitude vs X

This project was created to analyze weather patterns in relation to latitude throughout the world. Data was collected from the OpenWeatherMap API with a minimum sized dataset of 500 cities. This data was collected on December 5, 2019 and represents the weather conditions on that particular day.

We then utilized Matplotlib to plot 4 aspects of weather and compared this to the latitude. These factors included: Temperature, Humidity, Cloudiness and Wind Speed. All of the visualizations and source data can be found by navigating through this website. Explanations and descriptions of the visualizations are provided on each of the separate pages of analysis. Have fun!
